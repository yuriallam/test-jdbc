# Test JDBC

## Prerequisites
- Java
- MySQL

## Setup
- git clone the project
- Run the queries.sql file

## Usage
- Run the following command in the root directory `java -cp "MySQL ConnectorJ/8.0.25/mysql-connector-java-8.0.25.jar" src/Main.java`
